# jQuery Dynamic Sections - Course Projects

This project demonstrates dynamic section display using jQuery for four different courses: SDP, EDI, DT, and VIT.

## Features

- **Dynamic Section Switching**: Click navigation buttons to show/hide different course sections
- **Smooth Animations**: Fade in/out effects when switching between sections
- **Responsive Design**: Works on desktop and mobile devices
- **Interactive UI**: Hover effects and active states for buttons
- **jQuery Powered**: Uses jQuery for DOM manipulation and animations

## Course Sections

1. **SDP** - Software Development Process
2. **EDI** - Emerging Digital Industries
3. **DT** - Digital Transformation
4. **VIT** - Virtual and Immersive Technologies

## Technologies Used

- HTML5
- CSS3 (with gradients and animations)
- jQuery 3.7.1
- Responsive design principles

## How to Use

1. Open `index.html` in a web browser
2. Click on any course button (SDP, EDI, DT, VIT) to view that section
3. The active section will be highlighted and content will fade in smoothly

## jQuery Features Demonstrated

- `$(document).ready()` - DOM ready event
- `.click()` - Click event handling
- `.fadeIn()` / `.fadeOut()` - Animation effects
- `.addClass()` / `.removeClass()` - CSS class manipulation
- `.data()` - Data attribute access
- `.hover()` - Hover event handling
- `.css()` - Dynamic CSS manipulation
- `.each()` - Element iteration
- `.delay()` - Animation delay
