# Bootstrap Dynamic Tabs & Pills Website

A modern, responsive website demonstrating Bootstrap 5's dynamic tabs and pills navigation components.

## Features

### Tabs Navigation
- **Home Tab**: Welcome section with introduction
- **Profile Tab**: User profile with image and details
- **Services Tab**: Service offerings in card layout
- **Contact Tab**: Interactive contact form with validation

### Pills Navigation
- **Features Tab**: List of key features
- **Pricing Tab**: Pricing plans in card layout
- **Testimonials Tab**: Client testimonials

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Custom styling with gradients and animations
- **Bootstrap 5.3.0**: Responsive framework and components
- **JavaScript**: Dynamic interactions and form handling

## Key Highlights

✓ Fully responsive design
✓ Dynamic content switching without page reload
✓ Smooth animations and transitions
✓ Interactive form with validation
✓ Modern gradient color scheme
✓ Hover effects and visual feedback
✓ Cross-browser compatible

## How to Use

1. Open `index.html` in any modern web browser
2. Click on different tabs to see content change
3. Scroll down to see pills navigation
4. Try the contact form to see validation in action
5. Hover over elements to see interactive effects

## Customization

- **Colors**: Modify gradient colors in `styles.css`
- **Content**: Update tab/pill content in `index.html`
- **Behavior**: Adjust interactions in `script.js`
